# Social Media Marketing Performance Analysis 2024

Analyzed 5,600+ social media posts to identify key drivers of content 
performance for a global IT firm across 6 platforms and 8 regions.

## Tools
- **SQL** — data querying & aggregation
- **Python** (Pandas, Seaborn) — data cleaning & visualization  
- **Tableau** — interactive dashboard

## Key Findings
- Instagram leads all platforms at **15.85% ER** vs LinkedIn's 14.30%
- **Customer Story** content outperforms all categories at 20.22% ER — nearly 3x higher than Entertainment
- Peak posting time: **Monday 6PM** at 18.00% ER
- Organic delivers **4x more views** than Sponsored at equal engagement rate
